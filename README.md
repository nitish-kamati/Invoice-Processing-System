## 🚀 Project Overview

The **Invoice Processing System** is a full-stack web application that allows users to submit and manage invoice information through a simple web interface. The system processes invoice data using REST APIs and stores it securely in a MySQL database.

This project demonstrates the integration of a **React frontend**, **Spring Boot backend**, and **MySQL database**, simulating a real-world business workflow for invoice management.

## ⚙️ How It Works

1. The user fills in invoice information on the frontend interface.
2. The frontend sends the data to the backend using **Axios HTTP requests**.
3. Spring Boot receives the request through **REST API endpoints**.
4. The backend processes the data and stores it in the **MySQL database**.
5. A response is returned confirming successful invoice submission.
